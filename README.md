# Amazon Product Reviews - Data Analytics Project

This project was completed as part of the **CodeAlpha Data Analytics Internship**. It demonstrates end-to-end data analysis workflow by combining Exploratory Data Analysis (EDA), Natural Language Processing (NLP), and Data Visualization.

## 📌 Project Overview
The project analyzes customer reviews of Amazon products to understand customer satisfaction, rating distributions, and sentiments (Positive, Negative, Neutral).

## 🛠️ Tasks Completed
- **Task 1: Exploratory Data Analysis (EDA):** Analyzed data structure, checked missing values, calculated summary statistics for ratings, and engineered a new feature (`Review_Length`).
- **Task 2: Sentiment Analysis:** Used `TextBlob` (NLP library) to analyze the polarity of review texts and classify them into Positive, Negative, or Neutral categories.
- **Task 3: Data Visualization:** Created impactful visualizations including:
  - Overall Sentiment Distribution (Pie Chart)
  - Sentiment Breakdown by Product Ratings (Bar Plot)
  - Most Common Words in Reviews (WordCloud)

## 💻 Tech Stack Used
- **Language:** Python
- **Environment:** VS Code (Jupyter Notebook)
- **Libraries:** Pandas, NumPy, TextBlob, Matplotlib, Seaborn, WordCloud
